# Journal Selection & Submission Packs  -  Reference

## Ranking criteria (score each candidate, cite every fact)
| Criterion | What to check (live source) |
|-----------|------------------------------|
| Scope/fit | aims & scope page; recent similar articles (nursing workforce, satisfaction, retention, LMIC) |
| Indexing | PubMed/MEDLINE, Scopus, DOAJ listing |
| Cost | APC / open-access fee; any waiver for LMIC authors |
| Timeline | stated time-to-first-decision / to-publication |
| Impact | impact factor / CiteScore / quartile (with year and source) |
| Legitimacy | DOAJ membership, COPE membership, Think-Check-Submit; **screen out predatory venues** |

Every figure (APC, IF, timeline, indexing) must be cited to a live page  -  never asserted from memory. If a
fact cannot be verified live, say so and do not invent it.

## Candidate space (verify currency live before naming any)
Topical fits often include nursing-management, nursing, health-workforce, and public-health journals. Do not
rely on this list  -  confirm current scope, indexing, and standing by live search at run time.

## Submission pack (one per top-3 journal)
1. **Formatted manuscript**  -  restructured to the journal's current instructions for authors (section order,
   word and reference limits, abstract format, reference style); completed **STROBE checklist** attached.
2. **Cover-letter draft**  -  tailored to the journal's scope; states the contribution, confirms originality and
   no concurrent submission, and names suggested or excluded reviewers if the journal requests them.
3. **Submission checklist**  -  the journal's required items: ethics approval statement, consent, data
   availability statement, author contributions, conflicts of interest, funding, ORCID, reporting-guideline
   declaration (STROBE).

## Lock preservation
Every locked claim, number, table value, and citation is preserved exactly across all three packs. Formatting
changes never alter content. Confirm against `results_lock.json` / `protocol_lock.json` before delivering.

## Output
- journal-selection report (ranked three + cited rationale + a short recommendation)
- three submission packs (formatted manuscript + cover letter + submission checklist)
